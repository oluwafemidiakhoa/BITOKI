# BITfisher Logo and Images

## Required Images

To complete the BITfisher branding, add the following images to this directory:

### 1. Logo (bitfisher-logo.png)
- **Recommended size:** 200x200px or larger (PNG with transparency)
- **Description:** The blue and orange kingfisher bird logo
- **Usage:** Displayed in the navigation bar

### 2. Favicon (bitfisher-favicon.png)
- **Recommended size:** 32x32px or 64x64px
- **Description:** Small icon for browser tabs
- **Usage:** Shows in browser tab and bookmarks

## Logo Design Guidelines

**BITfisher Brand Colors:**
- Primary Blue: `#0EA5E9`
- Secondary Orange: `#F97316`
- Dark Navy: `#0F172A`

**Logo Elements:**
- Blue and orange kingfisher bird with wings spread
- Golden Bitcoin "B" emblem
- Dynamic, swift movement aesthetic

## How to Add

1. Save your logo image as `bitfisher-logo.png` in this directory
2. Save your favicon as `bitfisher-favicon.png` in this directory
3. Refresh your browser to see the changes

The application is already configured to use these images once they're added.
